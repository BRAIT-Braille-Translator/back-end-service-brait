# back-end-service-brait


## Getting Started

### Installation
1. clone the repo
   ```sh
   git clone https://github.com/BRAIT-Braille-Translator/back-end-service-brait.git
   ```
2. install npm package
   ```sh
   npm install
   ```
3. change example env to .env file
   ```sh
    cp .env.example .env
  ```
